# Vehicle Service Management System

A full-stack web application built using the MERN stack to streamline vehicle service operations. The system enables customers to book service appointments, technicians to manage assigned tasks, and administrators to oversee service requests through role-based dashboards.

## Features

- Secure role-based authentication (Admin, Technician, Customer)
- Online vehicle service booking
- Technician assignment and management
- Service status tracking
- Vehicle service history
- Responsive user interface
- RESTful API integration

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- HTML5
- CSS3
- JavaScript
